#Project Overview

This project performs Exploratory Data Analysis (EDA) on the Netflix Titles Dataset to understand trends related to content production, genres, countries, ratings, and audience targeting.

#Objectives

The analysis focuses on 5 key objectives:

Content Production vs Country

Content Production vs Year

Movies vs TV Shows Proportion

Genre Analysis

Rating Distribution (Audience Targeting)

#Data Cleaning

Before analysis, the following cleaning steps were applied:

Removed duplicate rows

Filled missing values

Replaced missing text fields with "Unknown"

Filled date_added and rating with their mode

Split genre column for detailed genre analysis
